AMD(Advanced Micro Devices, Inc.)，成立于1969年，3A构建方案(CPU、GPU、主板芯片组均由AMD制造提供)

AMD 相较Intel 的短板在跨核通讯，AMD 平台在同一个 CCX 内与 Intel 性能相近，跨 CCX 和跨 NUMA 性能会降低为 1/2 和 1/3。
